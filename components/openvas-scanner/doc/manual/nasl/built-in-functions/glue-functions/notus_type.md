# notus_type

## NAME

**notus_type** - Return the retained local Notus result type

## SYNOPSIS

*int* **notus_type**();

## DESCRIPTION

The Rust scanner's local feed-backed Notus implementation returns Skiron-style
`oid` and `message` dictionaries.

## RETURN VALUE

`1` for the retained result format.
